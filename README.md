# Book & Task Dashboard

A clean beginner-friendly web app that combines a book search tool with a simple task manager.

This project was built with HTML, CSS, and JavaScript. It uses the Open Library API to search books and displays results dynamically on the page.

## Features

- Search books using the Open Library API
- Display book title, author, and first publication year
- Add tasks to a study checklist
- Mark tasks as complete
- Delete tasks
- Responsive layout for desktop and mobile
- Clean semantic HTML structure
- Organized CSS custom properties
- JavaScript error handling with `try...catch`

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Fetch API
- Open Library API

## How to Run the Project

1. Download or clone the repository.
2. Open the folder in Visual Studio Code.
3. Right-click `index.html`.
4. Select **Open with Live Server**.

You can also open `index.html` directly in a browser, but Live Server is recommended.
